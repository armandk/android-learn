Class Overview

This class keeps track of the application's current activity. It is important to register this callback, otherwise the authentication interceptors will not be able to handle challenges properly! The recommended way to register this callback is by overriding android.app.Application and calling registerActivityLifecycleCallbacks from onCreate. 


 public class SampleApplication extends Application {
 @Override
 public void onCreate() {
 super.onCreate();
 registerActivityLifecycleCallbacks(AppLifecycleCallbackHandler.getInstance());
 }
 }

public boolean isInForeground ()

Returns true if the application is visible.

Note that if this method is called from another registered Application.ActivityLifecycleCallbacks method the return value will be unreliable since the order the Application.ActivityLifecycleCallbacks will be called is unknown. Instead, other Application.ActivityLifecycleCallbacks should use their own callback methods to track the application's state.

Returns
  • true if the application is visible, false otherwise.
