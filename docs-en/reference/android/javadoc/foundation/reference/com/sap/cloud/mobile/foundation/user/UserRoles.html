Class Overview

Loads information about the current user from mobile services. The information provides the name, id and roles of the current user. The id and the name of the user are always available. The `roles` parameter is available only if access control is enabled in the mobile services cockpit. Deprecated since 3.2, replaced with ERROR(/UserService)
