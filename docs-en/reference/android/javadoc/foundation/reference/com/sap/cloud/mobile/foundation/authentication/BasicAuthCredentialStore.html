public interface

BasicAuthCredentialStore

com.sap.cloud.mobile.foundation.authentication.BasicAuthCredentialStore
Known Indirect Subclasses
BasicAuthInMemoryStore, BasicAuthPersistentStore
BasicAuthInMemoryStore This class implements BasicAuthCredentialStore by simply keeping a hashmap in memory (there is no persistence across app restarts). 
BasicAuthPersistentStore This class implements BasicAuthCredentialStore by persisting credentials in SecureKeyValueStore  

Class Overview

This interface is used by BasicAuthDialogAuthenticator to store Basic Auth credentials to reuse on future challenges.

public abstract void deleteCredential (String rootUrl, String realm)

Deletes the stored credentials for a given root URL and realm.

Parameters
rootUrl The rootUrl for which the credentials should be deleted. The root URL is composed of the scheme, host, and realm.

public abstract String[] getCredential (String rootUrl, String realm)

Gets stored credentials for reuse.

Parameters
rootUrl The root URL for which the credentials are being requested for. The root URL is composed of the scheme, host, and port.
realm The realm for which the credentials are being requested for.
Returns
  • The stored credentials or null if none are stored for the given rootUrl and realm.

public abstract void storeCredential (String rootUrl, String realm, String[] credentials)

Stores credentials for future use.

Parameters
rootUrl The root URL for which the credentials should be used for. The root URL is composed of the scheme, host, and port.
realm The realm for which the credentials should be used for.
credentials This array should have a length of 2. The first index should have the username, the second index should have the password.
