package

com.sap.cloud.mobile.fiori.onboarding

Provides the view classes necessary for onboarding process. These views are meant to be lightweight alternatives to replace the old onboarding Activity counterparts. The general pattern is to include xxxScreen in your Activity or Fragment, customize the UI when necessary, and provide callbacks to handle button clicks.
