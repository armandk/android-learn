Class Overview

Settings object to configure the EnterPasscodeActivity with. An object of this type may be read from and written to either Intent objects. These settings can be used by the app developer to customize the enter passcode screen.

Alternatively, one may construct the settings object manually by using the default constructor and setting the fields individually.

It is recommended to read the documentation of the EnterPasscodeActivity and a PasscodeActionHandler to get a better insight into the various supported passcode related flows and how these screens operate.
