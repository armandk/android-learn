Class Overview

Settings object to configure the SetPasscodeActivity with. An object of this type may be read from and written to either Intent objects. These settings can be used by the app developer to customize the create passcode screen.

Alternatively, one may construct the settings object manually by using the default constructor and setting the fields individually.

It is recommended to read the documentation of the SetPasscodeActivity and a PasscodeActionHandler to get a better insight into the various supported passcode related flows and how these screens operate.
