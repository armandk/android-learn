Class Overview

Represents the screen settings for ConsentScreen. With the properties defined in this, ConsentScreen can be customized to the client needs, for example, change the button text for all buttons on the screen, add multiple screens with 'Next'/'Prev' buttons to navigate among the screens. If the content shown on the screen has multiple lines, with ConsentScreenSettings.ScreenType of FORCE_READ_ALL, 'Next' button will only be shown when all the content is read.

Get the screen type. 3 types are supported:

  • OPTIONAL: skip button will be shown
