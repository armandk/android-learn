getSubset

open fun getSubset(): MutableList<OfflineODataDefiningQuery!>!

The OfflineODataProvider will queue up download operations where a different subset of the defining queries is specified. The subset property of OfflineODataProviderDownloadProgress contains the subset of defining queries specified for the download operation. If no subset was specified, the subset property will be `null`.
