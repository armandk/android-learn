setCustomHeaders

fun setCustomHeaders(customHeaders: MutableMap<String!, String!>!): Unit

Defines the custom headers to be added to HTTP requests between the OfflineODataProvider and the server, and the server to the OData back end.

The keys are the header name. The values are header value.

Parameters

customHeaders - MutableMap<String!, String!>!: the custom HTTP headers. If null, an empty customer headers map is used.
