getPreviousUser

fun getPreviousUser(): String!

Get the previous user if the store is used with isForceUploadOnUserSwitch=true. It will be empty string if the local store was not opened with isForceUploadOnUserSwitch=true. For more information of isForceUploadOnUserSwitch, please refer to OfflineODataParameters.setForceUploadOnUserSwitch This function could be called whenever local store was opened or not.

Exceptions

OfflineODataException - if an error occurs getting user id

Return
String!: userId of user
