Sends the Offline OData store to sever.

Example Usage: 

// Send unecrypted store asynchronously
      offlineODataProvider.sendStore(() -> logger.info("Send store completed successfully."),
                                     (error) -> logger.error("Exception encountered in offlineOData send store:" + error.getMessage()));

Parameters

successHandler - Action0!: a success callback function.

failureHandler - Action1!: a failure callback function.

note - a note which is not exceeding 480 characters long.
