removeDefiningQuery

fun removeDefiningQuery(definingQuery: OfflineODataDefiningQuery!, successHandler: Action0!, failureHandler: Action1!): Unit

Removes a defining query.

Please reference #addDefiningQuery(OfflineODataDefiningQuery) for the three types of defining queries.

Before OfflineODataProvider is open:

  • The defining queries of type 1, 2 can only be removed for the first time.
  • The defining queries of type 3 can also be removed, these are not persistent, they will be lost the device is power off or application is terminated.

After OfflineODataProvider is open:

  • If 'allow_defining_query_removal' is not enabled, only the defining queries of type 3 can be removed, and failureHandler will be called with OfflineODataException for others.
  • If 'allow_defining_query_removal' is enabled, all types can be removed when there is no any pending change. Removing a defining query from the provider will delete all local data of the defining query in OfflineODataProvider like entities downloaded from the backend, relationships associated with the entities and their media streams. The local data is always deleted immediately, but if non-shared delta tracking is enabled, the name of the defining query and its change logs on the server will be deleted during next downloading.

Parameters

definingQuery - OfflineODataDefiningQuery!: the defining query to remove.

successHandler - Action0!: a success callback function.

failureHandler - Action1!: a failure callback function.

fun removeDefiningQuery(definingQuery: OfflineODataDefiningQuery!): Unit

Removes a defining query.

The defining queries of type 1 and 2 can only be removed before the OfflineODataProvider is open for the first time.

The defining queries of type 3 can be removed before or after the OfflineODataProvider has been opened.

Removing type 3 defining queries are not persistent when the OfflineODataProvider is not open, they will be lost the device is power off or application is terminated.

Note: This method does not support 'allow_defining_query_removal', please try #removeDefiningQuery(OfflineODataDefiningQuery, Action0, Action1)

Parameters

definingQuery - OfflineODataDefiningQuery!: the defining query to remove.

Exceptions

OfflineODataException - if an error occurs removing the defining query.
