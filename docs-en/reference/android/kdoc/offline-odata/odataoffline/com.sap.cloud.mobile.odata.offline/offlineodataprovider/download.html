fun download(successHandler: Action0!, failureHandler: Action1!): CancelToken!

Downloads the data for all defining queries from the OData back end to update the local data. If the defining queries are delta enabled, only the changes will be downloaded.

Example Usage: 

// Download data asynchronously
      offlineODataProvider.download(() -> logger.info("Download completed successfully."),
                                    (error) -> logger.error("Exception encountered in OfflineOData download: ", error));

Note: The downloading task may not back immediately since which is pending because of other concurrent downloadings or which is in a critical non-cancelled stage.

Parameters

successHandler - Action0!: a success callback function.

failureHandler - Action1!: a failure callback function.

Return
CancelToken!: a token to cancel the download operation

fun download(subset: MutableList<OfflineODataDefiningQuery!>!, successHandler: Action0!, failureHandler: Action1!): CancelToken!

Downloads the data for the defining queries from the OData back end to update the local data. If the defining queries are delta enabled, only the changes will be downloaded.

Example Usage: 

// Create subset of defining queries to download
      List< OfflineODataDefiningQuery > subset = new ArrayList<>();
      subset.add(new OfflineODataDefiningQuery("Customers", "Customers", false));
     
      // Download data asynchronously
      offlineODataProvider.download(subset,
                                    () -> logger.info("Download completed successfully."),
                                    (error) -> logger.error("Exception encountered in OfflineOData download: ", error));

Note: The downloading task may not back immediately since which is pending because of other concurrent downloadings or which is in a critical non-cancelled stage.

Parameters

subset - MutableList<OfflineODataDefiningQuery!>!: the subset of the defining queries to download.

successHandler - Action0!: a success callback function.

failureHandler - Action1!: a failure callback function.

Return
CancelToken!: a token to cancel the download operation
