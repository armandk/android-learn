updateFailedRequest

open fun updateFailedRequest(provider: OfflineODataProvider!, request: OfflineODataFailedRequest!): Unit

Called every time for a request that fails on OData backend during an upload.

Note that the upload operation will report a successful result even if some requests have failed against the OData backend.

Parameters

provider - OfflineODataProvider!: the OfflineODataProvider which uploading changes.

request - OfflineODataFailedRequest!: the information of the failed request.
