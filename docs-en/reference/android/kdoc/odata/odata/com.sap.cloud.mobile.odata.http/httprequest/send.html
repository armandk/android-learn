send

open fun send(): Unit

Send this HTTP request to the server, and wait for a response. If HttpRequest.streamResponse is true, then waiting is only until response headers come back, and response content may be streamed as it is accessed. If an application calls this function, then it must also call HttpRequest.close after accessing all desired response content, to avoid resource leaks.
