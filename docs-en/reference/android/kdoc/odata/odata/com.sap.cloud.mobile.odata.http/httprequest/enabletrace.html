enableTrace

open fun enableTrace(@Nullable @Nullable serviceName: @Nullable String?, traceRequest: Boolean, traceHeaders: Boolean, traceContent: Boolean, prettyPrint: Boolean): Unit

Enable HTTP request tracing. Call this before calling HttpRequest.send.

Parameters

serviceName - String?: (nullable) Name of the service which this request will be sent to.

traceRequest - Boolean: Trace the request (URL), and basic request/response flow. Note: this may result in tracing of sensitive data, such as URL query parameters.

traceHeaders - Boolean: Trace the headers. Note: this may result in tracing of sensitive data, such as authentication information or cookie values.

traceContent - Boolean: Trace the content. Note: this may result in tracing of sensitive data.

prettyPrint - Boolean: If traceContent is true and HttpRequest.streamResponse is false, then content tracing of JSON or XML will be pretty-printed.
