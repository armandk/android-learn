resolveNamespaces

open fun resolveNamespaces(): Unit

Traverse the element hierachy starting at the XmlDocument.rootElement, ensuring that every node has the XmlElement.namespaceURI set according to the "xmlns" attributes and namespace prefixes encountered in the nodes.
