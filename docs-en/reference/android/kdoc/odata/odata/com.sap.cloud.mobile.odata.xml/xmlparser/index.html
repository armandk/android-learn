XmlParser

open class XmlParser

Parser for XML. Supports parsing into an com.sap.cloud.mobile.odata.xml.XmlDocument or com.sap.cloud.mobile.odata.xml.XmlElement. Parsing options are complete (XmlParser.parseDocument, XmlParser.parseElement) and streamed (XmlParser.startDocument, XmlParser.startElement).
