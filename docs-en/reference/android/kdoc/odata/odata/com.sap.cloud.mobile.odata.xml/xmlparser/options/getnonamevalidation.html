getNoNameValidation

open fun getNoNameValidation(): Boolean

Return set this to true to disable validation of element/attribute names. This may result in a performance improvement during parsing, although it may permit some invalid documents to be successfully parsed.

Return
Boolean: Set this to true to disable validation of element/attribute names.
