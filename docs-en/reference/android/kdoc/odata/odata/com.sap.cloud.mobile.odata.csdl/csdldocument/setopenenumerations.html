setOpenEnumerations

open fun setOpenEnumerations(value: Boolean): Unit

Set allows addition of enumeration members in the latest service metadata before calling DataService.refreshMetadata. Defaults to false. Warning: enabling this option can result in application instability if the application uses proxy classes without "-open:enumerations".

Parameters

value - Boolean: Allows addition of enumeration members in the latest service metadata before calling DataService.refreshMetadata. Defaults to false.
