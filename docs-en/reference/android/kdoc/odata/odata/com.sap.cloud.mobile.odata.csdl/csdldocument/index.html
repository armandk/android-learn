open fun addVocabularyReference(ns: String, uri: String?, alias: String?): Unit

Add a well-known vocabulary reference to this metadata. If this metadata already has a reference to the well-known vocabulary, then no change is made. If the reference is to one of the CsdlDocument.builtinVocabularyNamespaces, then all defined annotation terms/types from the vocabulary will also be added to the current metadata.
