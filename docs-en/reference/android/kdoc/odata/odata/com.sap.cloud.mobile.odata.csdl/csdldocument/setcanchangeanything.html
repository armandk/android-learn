setCanChangeAnything

open fun setCanChangeAnything(value: Boolean): Unit

Set allows any model element to be changed in the latest service metadata before calling DataService.refreshMetadata. Defaults to false. Warning: enabling this option can result in application instability (depending on the changes) if the application uses proxy classes!

Parameters

value - Boolean: Allows any model element to be changed in the latest service metadata before calling DataService.refreshMetadata. Defaults to false.
