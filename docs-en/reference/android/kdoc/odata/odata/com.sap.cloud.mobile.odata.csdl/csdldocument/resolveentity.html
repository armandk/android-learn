resolveEntity

open fun resolveEntity(@NonNull @NotNull entity: @NotNull EntityValue): Unit

Resolve the EntityValue.entitySet of `entity' (if it is not already set).

Throws:

com.sap.cloud.mobile.odata.DataServiceException if entity.entitySet is not set, and is not unique (i.e. multiple entity sets use the same entity type)

or there is a circular dependency in related entities (parent entity is set as child as well) or an entity is resolved multiple times meaning that it is embedded as child more than once.

Parameters

entity - EntityValue: Entity to resolve.
