getCanRemoveAnything

open fun getCanRemoveAnything(): Boolean

Return allows any model element to be removed in the latest service metadata before calling DataService.refreshMetadata. Defaults to false. Warning: enabling this option can result in application instability unless the application checks isRemoved flags on model elements!

Return
Boolean: Allows any model element to be removed in the latest service metadata before calling DataService.refreshMetadata. Defaults to false.
