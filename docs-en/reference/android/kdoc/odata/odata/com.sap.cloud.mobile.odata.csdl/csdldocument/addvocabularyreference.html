open fun addVocabularyReference(@NonNull @NotNull ns: @NotNull String, @Nullable @Nullable uri: @Nullable String?, @Nullable @Nullable alias: @Nullable String?): Unit

Add a well-known vocabulary reference to this metadata. If this metadata already has a reference to the well-known vocabulary, then no change is made. If the reference is to one of the CsdlDocument.builtinVocabularyNamespaces, then all defined annotation terms/types from the vocabulary will also be added to the current metadata.

See Also:

CsdlDocument.builtinVocabularyNamespaces.

Parameters

ns - String: CSDL namespace.

uri - String?: (nullable) Optional namespace URI. For builtin vocabulary namespaces, this can be derived from the namespace. For references to non-builtin vocabularies, it must be provided.

alias - String?: (nullable) Optional namespace alias. For builtin vocabulary namespaces, this can be derived from the namespace. For references to non-builtin vocabularies, it can be provided.
