hasOpenEnumerations

open fun hasOpenEnumerations(): Boolean

Return allows addition of enumeration members in the latest service metadata before calling DataService.refreshMetadata. Defaults to false. Warning: enabling this option can result in application instability if the application uses proxy classes without "-open:enumerations".

Return
Boolean: Allows addition of enumeration members in the latest service metadata before calling DataService.refreshMetadata. Defaults to false.
