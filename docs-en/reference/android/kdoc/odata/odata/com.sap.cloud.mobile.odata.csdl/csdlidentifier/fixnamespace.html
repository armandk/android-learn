fixNamespace

@NonNull @NotNull open static fun fixNamespace(@NonNull @NotNull name: @NotNull String): @NotNull String

Fix an invalid OData namespace (if it is invalid). Forward slashes will be changed to ".". Invalid leading/trailing characters will be removed. All other invalid characters will be changed to "_".

Parameters

name - String: Namespace identifier.

Return
String: Fixed namespace (or original identifier, if it was already valid).
