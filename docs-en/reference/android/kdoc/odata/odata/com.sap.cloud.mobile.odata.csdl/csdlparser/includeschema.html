includeSchema

open fun includeSchema(@NonNull @NotNull schema: @NotNull DataSchema): Unit

Include the definitions of a pre-parsed schema into the document to be parsed. Call this function before CsdlParser.parse if the document to be parsed has an edmx:Reference element for a well-known schema, and the pre-parsed schema should be used rather than loading the referenced schema over the network.

Parameters

schema - DataSchema: Pre-parsed schema.
