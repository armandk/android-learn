setCsdlFetcher

open fun setCsdlFetcher(@Nullable @Nullable value: @Nullable CsdlFetcher?): Unit

Set a callback which will be used to resolve any CSDL Reference that is encountered during parsing, in order to fetch the referenced metadata. Must be set before CsdlParser.parse is called, or else any CSDL Reference that is encountered during parsing will result in a com.sap.cloud.mobile.odata.csdl.CsdlException being thrown.

value - CsdlFetcher?: A callback which will be used to resolve any CSDL Reference that is encountered during parsing, in order to fetch the referenced metadata.
