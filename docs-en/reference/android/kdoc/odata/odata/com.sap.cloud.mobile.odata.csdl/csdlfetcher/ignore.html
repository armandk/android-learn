ignore

open fun ignore(@NonNull @NotNull uri: @NotNull String, @NonNull @NotNull ns: @NotNull String): Boolean

Return true if this fetcher wants to force the com.sap.cloud.mobile.odata.csdl.CsdlParser to ignore a reference. This function returns false by default. Override this function in subclasses to return true only for those well-known location/namespace values that the fetcher wants to suppress fetching for.

Parameters

uri - String: Location of document to be read.

ns - String: Namespace for schema to be read.

Return
Boolean: true if this fetcher wants to force the com.sap.cloud.mobile.odata.csdl.CsdlParser to ignore a reference.
