accept

 open fun accept(uri: String, ns: String): Boolean

Return true if this fetcher accepts a location (or associated namespace), meaning that a CsdlFetcher.fetch call would be expected to succeed. This function returns false by default for OData standard vocabulary namespaces (Org.OData.*), otherwise this function returns true. Override this function in subclasses to return true only for those well-known location/namespace values that the fetcher expects that it can resolve to a service metadata document.
