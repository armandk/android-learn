normalize

@NonNull @NotNull open fun normalize(): @NotNull GlobalDateTime

Normalize this dateTime value to UTC. For example "2000-01-01T14:00:00+12:00" would normalize to "2000-01-01T02:00:00Z"

Return
GlobalDateTime: An equivalent value with all fields normalized (adjusted) to UTC, such that GlobalDateTime.offset is zero.
