getCacheMetadata

open fun getCacheMetadata(): Boolean

Return should loaded service metadata be cached in a local file? If ServiceOptions.metadataFile is non-null, then the cache file name will be ServiceOptions.metadataFile with ".cache" appended. If ServiceOptions.metadataFile is null, then the cache file name will be the service name with ".csdl.cache" appended.
