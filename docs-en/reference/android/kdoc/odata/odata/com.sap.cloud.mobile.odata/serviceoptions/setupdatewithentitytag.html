setUpdateWithEntityTag

open fun setUpdateWithEntityTag(value: Boolean): Unit

Set if the backend system returns an ETag header when querying for an entity (or an "etag" annotation when querying for multiple entities), should updates of the entity send the ETag value with an If-Match header? Defaults to true.

Parameters

value - Boolean: If the backend system returns an ETag header when querying for an entity (or an "etag" annotation when querying for multiple entities), should updates of the entity send the ETag value with an If-Match header? Defaults to true.
