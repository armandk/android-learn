setCacheMetadata

open fun setCacheMetadata(value: Boolean): Unit

Set should loaded service metadata be cached in a local file? If ServiceOptions.metadataFile is non-null, then the cache file name will be ServiceOptions.metadataFile with ".cache" appended. If ServiceOptions.metadataFile is null, then the cache file name will be the service name with ".csdl.cache" appended.

See Also:

DataService.loadMetadata.

Parameters

value - Boolean: Should loaded service metadata be cached in a local file?
