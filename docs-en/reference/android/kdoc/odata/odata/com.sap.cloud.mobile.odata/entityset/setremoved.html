setRemoved

open fun setRemoved(value: Boolean): Unit

Set has this model element been removed by a successful call to DataService.refreshMetadata? If `isRemoved == true, that implies that the old metadata (before refreshMetadata was called) did include this model element, and that the new metadata (after refreshMetadata was called) does not include this model element.

See Also:

EntitySet.canBeRemoved.

Parameters

value - Boolean: Has this model element been removed by a successful call to `DataService.refreshMetadata`?
