setCanBeRemoved

open fun setCanBeRemoved(value: Boolean): Unit

Set can this model element be removed by DataService.refreshMetadata. Defaults to false. Set this to true to indicate that the application is coded to be able able to function correctly without this model element.

value - Boolean: Can this model element be removed by `DataService.refreshMetadata`. Defaults to `false`.
