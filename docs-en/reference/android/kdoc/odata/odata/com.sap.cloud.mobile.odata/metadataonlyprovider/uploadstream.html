uploadStream

open fun uploadStream(@NonNull @NotNull entity: @NotNull EntityValue, @NonNull @NotNull link: @NotNull StreamLink, @NonNull @NotNull content: @NotNull StreamBase, @NonNull @NotNull headers: @NotNull HttpHeaders, @NonNull @NotNull options: @NotNull RequestOptions): Unit

Overrides DataServiceProvider.uploadStream

Upload content for a stream property to the target system. Caution: Having too many threads simultaneously uploading streams may result in out-of-memory conditions on memory-constrained devices.

See Also:

Property.getStreamLink.

Parameters

entity - EntityValue: Entity containing the stream property whose content is to be uploaded.

link - StreamLink: Stream link for the stream to be uploaded.

content - StreamBase: Upload stream content. Will be closed before this function returns.

headers - HttpHeaders: Request-specific headers.

options - RequestOptions: Request-specific options.
