login

open fun login(@NonNull @NotNull username: @NotNull String, @NonNull @NotNull password: @NotNull String): Unit

Set the shared Authorization header to be used for HTTP/HTTPS with pre-emptive BASIC authentication. If credentials are required by the remote data service, they must be set before any function calls (such as OnlineODataProvider.loadMetadata) which may make a network request. For services which require challenge-based authentication, a request interceptor/observer should be used.

See Also:

OnlineODataProvider.sharedHeaders.

Parameters

username - String: Client's username.

password - String: Client's password.
