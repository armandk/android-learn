shareNullable

@NonNull @NotNull open static fun shareNullable(@NonNull @NotNull list: @NotNull ListBase): @NotNull DataValueList

Return a new DataValueList that shares the ListBase.untypedList as the list parameter, including nulls. To ensure type safety, items in list that do not have the item type will be removed.

Parameters

list - ListBase: List whose items will be shared by the resulting list.

Return
DataValueList: A new list of item type com.sap.cloud.mobile.odata.DataValue, sharing the same items as list.
