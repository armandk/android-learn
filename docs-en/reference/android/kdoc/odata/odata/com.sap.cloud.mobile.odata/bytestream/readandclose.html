readAndClose

@NonNull @NotNull open fun readAndClose(): ByteArray

Read all remaining data until the end of the stream, then close this stream. Caution: this function loads all stream content into memory at once. Consider reading a chunk-at-a-time in a loop, using ByteStream.readBinary.

Return
ByteArray: The remaining data.
