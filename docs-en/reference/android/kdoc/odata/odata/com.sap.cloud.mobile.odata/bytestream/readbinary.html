@Nullable @Nullable open fun readBinary(length: Int): ByteArray?

Read up to length bytes from this stream (may read less, even if the end of stream is not reached).

Parameters

length - Int: Maximum number of bytes to read (must be greater than zero). Defaults to 20,000.

Return
ByteArray?: (nullable) Binary value with at most length bytes, or null if there is no remaining data in the stream.
