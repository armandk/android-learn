fun setDefaultValues(nestedComplex: Boolean, nestedEntity: Boolean, defaultOptional: Boolean, defaultRequired: Boolean, skipConstraints: Boolean): Unit

Recursively apply default values to any unset properties of this structure. If a property has a model-defined default value, it will always be applied (since that default value is part of a server's contract with the client, it must always be valid to apply it). If the property doesn't have a model-defined default value, then null will be used for nullable properties, and false/empty/zero values will be applied otherwise as appropriate. Dates will default to 2000-01-01.

See Also:

StructureBase.hasDataValue, Property.defaultValue, StructureType.structuralProperties, StructureType.navigationProperties.

Parameters

nestedComplex - Boolean: Apply recursively to complex-typed (structural) properties of this structure? Defaults to true.

nestedEntity - Boolean: Apply recursively to entity-typed (navigation) properties of this structure? Defaults to false.

defaultOptional - Boolean: Apply non-model-defined default values to optional (nullable) properties?

defaultRequired - Boolean: Apply non-model-defined default values to required (non-nullable) properties?

skipConstraints - Boolean: Avoid applying default values to properties which have a database-level default constraint.
