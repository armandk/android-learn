setUnsafeValue

open fun setUnsafeValue(@NonNull @NotNull property: @NotNull Property, @Nullable @Nullable value: @Nullable DataValue?): Unit

Set the value of a specified property (like StructureBase.setDataValue), but bypassing the usual validation checks. This may be useful in performance critical code, but note that it will be assumed that:

  • The property belongs to the structure's type.
  • The value has the correct type for the property.

Parameters

property - Property: The specified property.

value - DataValue?: (nullable) The property value (or null), using the appropriate subclass of com.sap.cloud.mobile.odata.DataValue.
