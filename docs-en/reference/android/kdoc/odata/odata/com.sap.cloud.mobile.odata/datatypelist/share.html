@NonNull @NotNull open static fun share(@NonNull @NotNull list: @NotNull ListBase): @NotNull DataTypeList

Return a new com.sap.cloud.mobile.odata.DataTypeList that shares the ListBase.untypedList as the list parameter. To ensure type safety, items in list that do not have the item type com.sap.cloud.mobile.odata.DataType will be removed.

Parameters

list - ListBase: List whose items will be shared by the resulting list.

Return
DataTypeList: A new list of item type com.sap.cloud.mobile.odata.DataType, sharing the same items as list.
