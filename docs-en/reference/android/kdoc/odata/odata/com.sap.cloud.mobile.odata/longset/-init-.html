LongSet(capacity: Int)

Construct a new set with LongSet.size of zero and optional initial capacity. A set can expand in size beyond its initial capacity, but best performance will be obtained if the initial capacity is close to the set's maximum size.

Parameters

capacity - Int: Optional initial capacity.
