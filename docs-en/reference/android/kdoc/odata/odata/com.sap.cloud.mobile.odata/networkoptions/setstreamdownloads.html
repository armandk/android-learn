setStreamDownloads

open fun setStreamDownloads(value: Boolean): Unit

Set should downloads be streamed? Set this to false to force the use of non-streamed downloads. True by default.

See Also:

DataService.downloadMedia, DataService.downloadStream.

Parameters

value - Boolean: Should downloads be streamed? Set this to `false` to force the use of non-streamed downloads.
