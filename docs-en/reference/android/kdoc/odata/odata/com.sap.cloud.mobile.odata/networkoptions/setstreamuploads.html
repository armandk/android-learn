setStreamUploads

open fun setStreamUploads(value: Boolean): Unit

Set should uploads be streamed? Set this to false to force the use of non-streamed uploads. True by default.

See Also:

DataService.createMedia, DataService.uploadMedia, DataService.uploadStream.

Parameters

value - Boolean: Should uploads be streamed? Set this to `false` to force the use of non-streamed uploads.
