executeMethod

@Nullable @Nullable abstract fun executeMethod(@NonNull @NotNull method: @NotNull DataMethod, @NonNull @NotNull parameters: @NotNull ParameterList, @NonNull @NotNull headers: @NotNull HttpHeaders, @NonNull @NotNull options: @NotNull RequestOptions): @Nullable DataValue?

Execute a data method (action or function) in the target system. Actions may have backend side-effects. Functions should not have backend side-effects.

Throws:

com.sap.cloud.mobile.odata.DataServiceException or com.sap.cloud.mobile.odata.DataNetworkException if an error occurs during action invocation.

Parameters

method - DataMethod: Data method.

parameters - ParameterList: Method parameters.

headers - HttpHeaders: Request-specific headers.

options - RequestOptions: Request-specific options.

Return
DataValue?: (nullable) The method result, or `null` if the method has no result.
