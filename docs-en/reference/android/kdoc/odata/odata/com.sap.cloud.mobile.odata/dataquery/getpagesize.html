getPageSize

@Nullable @Nullable open fun getPageSize(): @Nullable Int?

Return (nullable) The maximum number of results per page. Note that this is just a hint to the service. The service is permitted to ignore the client-requested page size. Use DataQuery.skipCount and DataQuery.topCount for predictable paging.

Return
Int?: (nullable) The maximum number of results per page.
