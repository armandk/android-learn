getSparseArray

open fun getSparseArray(): Boolean

Return if true, the query result entities will use sparse array storage of properties, to conserve memory. This is only applicable to queries whose result entities will not be updated.

See Also:

DataQuery.sparse.

Return
Boolean: If `true`, the query result entities will use [sparse array](https://en.wikipedia.org/wiki/Sparse_matrix) storage of properties, to conserve memory.
