setSparseArray

open fun setSparseArray(value: Boolean): Unit

Set if true, the query result entities will use sparse array storage of properties, to conserve memory. This is only applicable to queries whose result entities will not be updated.

See Also:

DataQuery.sparse.

Parameters

value - Boolean: If `true`, the query result entities will use [sparse array](https://en.wikipedia.org/wiki/Sparse_matrix) storage of properties, to conserve memory.
