setUnboundMethod

open fun setUnboundMethod(@Nullable @Nullable value: @Nullable String?): Unit

Set qualified name of the associated unbound method. For OData V4+ ActionImport, this is the name of the corresponding Action. For OData V4+ FunctionImport, this is the name of the corresponding Function. Otherwise null.
