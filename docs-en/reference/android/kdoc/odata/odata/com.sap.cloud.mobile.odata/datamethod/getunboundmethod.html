getUnboundMethod

@Nullable @Nullable open fun getUnboundMethod(): @Nullable String?

Return (nullable) Qualified name of the associated unbound method. For OData V4+ ActionImport, this is the name of the corresponding Action. For OData V4+ FunctionImport, this is the name of the corresponding Function. Otherwise null.

String?: (nullable) Qualified name of the associated unbound method.
