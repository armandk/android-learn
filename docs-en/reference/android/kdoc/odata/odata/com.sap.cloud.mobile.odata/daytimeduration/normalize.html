@NonNull @NotNull open fun normalize(): @NotNull DayTimeDuration

Return an equivalent value with all fields normalized. The result value will have nanos < 1000000000, seconds < 60, minutes < 60, hours < 24. The result value will be DayTimeDuration.equal to this value.
