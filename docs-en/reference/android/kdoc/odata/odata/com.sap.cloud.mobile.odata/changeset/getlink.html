getLink

@NonNull @NotNull open fun getLink(index: Int): @NotNull ChangedLink

Return the changed link, if isLink(index) is true, otherwise throws undefined. The ChangedLink.isCreated, ChangedLink.isUpdated and ChangedLink.isDeleted properties can be accessed on the resulting changed link to determine the type of change.

Example:

      public void showChangedLinks(final ChangeSet changes) {
          int n = changes.size(); {
              int i = 0;
              for (; (i < n); i++) {
                  if (changes.isLink(i)) {
                      ChangedLink change = changes.getLink(i);
                      if (change.isCreated()) {
                          Example.show("created link");
                      }
                      else if (change.isUpdated()) {
                          Example.show("updated link");
                      }
                      else if (change.isDeleted()) {
                          Example.show("deleted link");
                      }
                  }
              }
          }
      }

Parameters

index - Int: From zero to size - 1.

Return
ChangedLink: The changed link, if isLink(index) is true, otherwise throws undefined.
