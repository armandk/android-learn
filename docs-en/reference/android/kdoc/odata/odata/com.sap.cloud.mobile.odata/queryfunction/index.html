QueryFunction

open class QueryFunction

Represents a OData query function. Query functions are typically used within DataQuery.filter. com.sap.cloud.mobile.odata.QueryFunctionCall can be constructed using static calls on com.sap.cloud.mobile.odata.QueryFunction or, more conveniently, by chained calls on com.sap.cloud.mobile.odata.QueryValue.

Example using proxy classes (with QueryFunction static calls):

  public void queryFunctionExample() {
      NorthwindService service = this.getService();
      DataQuery query = new DataQuery()
          .filter(QueryFunction.contains(QueryFunction.toLower(Customer.address),
              StringValue.of("king")));
      List<Customer> customers = service.getCustomers(query);
      this.showCustomers(customers);
  }
Example using proxy classes (with QueryValue chained calls):

  public void queryValueFunctionExample() {
      NorthwindService service = this.getService();
      DataQuery query = new DataQuery()
          .filter(Customer.address.toLower().contains("king"));
      List<Customer> customers = service.getCustomers(query);
      this.showCustomers(customers);
  }
