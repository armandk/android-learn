readAndClose

@NonNull @NotNull open fun readAndClose(): @NotNull String

Read all remaining text until the end of this stream, then close this stream. Caution: this function loads all stream content into memory at once. Consider reading a chunk-at-a-time in a loop, using CharStream.readString.

Return
String: The remaining text.
