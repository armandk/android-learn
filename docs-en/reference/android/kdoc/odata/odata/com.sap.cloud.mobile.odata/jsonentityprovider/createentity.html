createEntity

open fun createEntity(@NonNull @NotNull entity: @NotNull EntityValue, @NonNull @NotNull headers: @NotNull HttpHeaders, @NonNull @NotNull options: @NotNull RequestOptions): Unit

Overrides DataServiceProvider.createEntity

Create an entity in the target system. Automatically calls CsdlDocument.resolveEntity to ensure that EntityValue.entitySet is available.

Throws:

com.sap.cloud.mobile.odata.DataServiceException if the entity set hasn't been explicitly provided before calling createEntity and there isn't a unique entity set for the entity type.

See Also:

EntityValue.ofType, EntityValue.inSet.

Parameters

entity - EntityValue: Entity to be created.

headers - HttpHeaders: Request-specific headers.

options - RequestOptions: Request-specific options.
