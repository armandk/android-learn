Overrides DataServiceProvider.uploadMedia

Upload content for a media entity to the target system Caution: Having too many threads simultaneously uploading streams may result in out-of-memory conditions on memory-constrained devices. Note: this function cannot be used to create a media entity. See DataService.createMedia.

Parameters

entity - EntityValue: Entity whose content is to be uploaded.

content - StreamBase: Upload stream content. Will be closed before this function returns.

headers - HttpHeaders: Request-specific headers.

options - RequestOptions: Request-specific options.
