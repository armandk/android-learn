setMustBeModified

open fun setMustBeModified(value: Boolean): Unit

Set true if the entity must be modified by an update request, even if it is apparently unchanged. This ensures that DataService.updateEntity will force an update in the backend database/system.

Parameters

value - Boolean: true if the entity must be modified by an update request, even if it is apparently unchanged.
