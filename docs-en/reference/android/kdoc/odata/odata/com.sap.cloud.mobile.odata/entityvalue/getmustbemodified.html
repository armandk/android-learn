getMustBeModified

open fun getMustBeModified(): Boolean

Return true if the entity must be modified by an update request, even if it is apparently unchanged. This ensures that DataService.updateEntity will force an update in the backend database/system.

Return
Boolean: true if the entity must be modified by an update request, even if it is apparently unchanged.
