setInErrorState

open fun setInErrorState(value: Boolean): Unit

Set true if this entity is in error state, for example due to the failed backend execution of an uploaded offline create/update/delete operation. If true, then one of EntityValue.isCreated, EntityValue.isUpdated or EntityValue.isDeleted should also be true to indicate the kind of failed operation. In some cases (e.g. system unable to distinguish create from update due to "upsert" behaviour) then EntityValue.isCreated, EntityValue.isUpdated and EntityValue.isDeleted might all be false.

Parameters

value - Boolean: true if this entity is in error state, for example due to the failed backend execution of an uploaded offline create/update/delete operation.
