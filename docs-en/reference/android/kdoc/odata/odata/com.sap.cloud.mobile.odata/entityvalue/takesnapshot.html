takeSnapshot

open fun takeSnapshot(): Unit

See takeSnapshot(boolean).

open fun takeSnapshot(canDefer: Boolean): Unit

Remember a snapshot of the values of structural/dynamic properties of this entity in EntityValue.oldEntity. Snapshots are used to detect which properties have changed when an update is to be issued.

See Also:

DataType.isMutable.

Parameters

canDefer - Boolean: Can the snapshot be deferred? This can improve query performance if queried entities are not subsequently updated. Defaults to `false`. Should only be `true` if all selected properties are immutable.
