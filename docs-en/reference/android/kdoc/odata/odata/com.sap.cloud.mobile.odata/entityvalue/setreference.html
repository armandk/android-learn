setReference

open fun setReference(value: Boolean): Unit

Set true if this object is an entity reference. If true, the EntityValue.editLink and/or EntityValue.readLink will be non-null, or the key properties will be available. Non-key properties will not be available.

Parameters

value - Boolean: true if this object is an entity reference. If true, the EntityValue.editLink and/or EntityValue.readLink will be non-null, or the key properties will be available.
