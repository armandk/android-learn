DataMethodCall()

Encapsulates the application of an OData method (function or action) to a list of parameters. This is used to represent calls to functions and actionn defined within an entity model, as opposed to com.sap.cloud.mobile.odata.QueryFunction which is used to represent calls to builtin functions.
