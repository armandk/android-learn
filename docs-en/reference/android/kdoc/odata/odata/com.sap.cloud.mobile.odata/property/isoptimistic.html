isOptimistic

open fun isOptimistic(): Boolean

Return true if this property is used for optimistic concurrency control (part of entity tag). For OData V3, this relates to a ConcurrencyMode of "Fixed". For OData V4, this relates to the use of Org.OData.Core.V1.OptimisticConcurrency in one or more entity sets using the entity type containing this property.

Return
Boolean: true if this property is used for optimistic concurrency control (part of entity tag).
