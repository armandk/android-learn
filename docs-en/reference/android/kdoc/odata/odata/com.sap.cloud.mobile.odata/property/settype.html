setType

open fun setType(@NonNull @NotNull value: @NotNull DataType): Unit

Set synonym for Property.dataType. If this property has a basic type, then type.isBasic will be true. If this property has a basic list type, then type.isBasicList will be true. If this property has a complex type, then type.isComplex will be true and type will be an instance of com.sap.cloud.mobile.odata.ComplexType. If this property has a complex list type, then type.isComplexList will be true and type.itemType will be an instance of com.sap.cloud.mobile.odata.ComplexType. If this property has an entity type, then type.isEntity will be true and type will be an instance of com.sap.cloud.mobile.odata.EntityType. If this property has an entity list type, then type.isEntityList will be true and type.itemType will be an instance of com.sap.cloud.mobile.odata.EntityType.

See Also:

DataType.isBasic, DataType.isBasicList, DataType.isComplex, DataType.isComplexList, DataType.isEntity, DataType.isEntityList.
