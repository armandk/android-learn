next

open fun next(): Boolean

Overrides DeltaStream.next

If this stream has not been DeltaStream.close and has a next delta item, then advance to the next delta item and return true. Otherwise return false.

See Also:

DeltaStream.hasCurrent, DeltaStream.hasEntity, DeltaStream.hasLink, DeltaStream.getCurrent, DeltaStream.getEntity, DeltaStream.getLink.

Return
Boolean: `true` if there is a next delta item available.
