decompress

@NonNull @NotNull open fun decompress(): @NotNull JsonElementStream

Enable decompression, where the document may contain an outermost object whose first field has name "$" and whose value is a compression dictionary expressed as an array of strings. Elsewhere in the document, a field name "$N" (where N is an integer >= 0) will be treated as a shorthand for the correponding name in the compression dictionary. Example:

{"$": "ABC", "XYZ", "value": {"$0": 123}, {"$1": 789}}
      is decompressed as {"value": {"ABC": 123}, {"XYZ": 789}}

Return
JsonElementStream: This element stream, with decompression enabled.
