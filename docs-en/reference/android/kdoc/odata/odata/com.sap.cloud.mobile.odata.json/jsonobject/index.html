JsonOrderedObject

 open class JsonOrderedObject : JsonObject

A specialization of com.sap.cloud.mobile.odata.json.JsonObject which retains the ordering of entries. While the JSON specification defined JSON objects as unordered, some uses of JSON may assume that processors can retain entry order. Performance Note: the time complexity of JsonOrderedObject.delete is O(n), rather than O(log(n)) for com.sap.cloud.mobile.odata.json.JsonObject.
