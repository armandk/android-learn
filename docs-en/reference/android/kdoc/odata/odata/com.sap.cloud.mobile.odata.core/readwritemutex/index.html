ReadWriteMutex

open class ReadWriteMutex

Allows the implementation of concurrent access for read-only operations, while write operations require exclusive access.

See Also:

Readers-writer lock.

Note: this implementation is not reentrant. If a thread tries to acquire a lock on this mutex while already holding a lock, the thread will hang.
