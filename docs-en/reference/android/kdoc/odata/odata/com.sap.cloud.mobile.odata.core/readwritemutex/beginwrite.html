beginWrite

open fun beginWrite(): Unit

Acquire an exclusive write lock, which can be released using ReadWriteMutex.endRead.

See Also:

ReadWriteMutex.writeLock which, if used instead of pairing beginWrite / endWrite calls, guarantees that the write lock is released when the action is complete.
