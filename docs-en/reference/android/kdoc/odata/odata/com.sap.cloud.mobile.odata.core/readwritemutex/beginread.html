beginRead

open fun beginRead(): Unit

Acquire a shared read lock, which can be released using ReadWriteMutex.endRead.

See Also:

ReadWriteMutex.writeLock which, if used instead of pairing beginRead / endRead calls, guarantees that the read lock is released when the action is complete.
