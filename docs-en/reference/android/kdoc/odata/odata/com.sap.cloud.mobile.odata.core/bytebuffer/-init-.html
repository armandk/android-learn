ByteBuffer(capacity: Int)

Construct a new buffer with a ByteBuffer.length of zero and specified initial capacity. A buffer can expand in length beyond its initial capacity, but best performance will be obtained if the initial capacity is close to the buffer's maximum length.

Parameters

capacity - Int: Initial capacity. Defaults to 16.
