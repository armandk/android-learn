setRange

fun setRange(index: Int, @NonNull @NotNull value: ByteArray): Unit

Copy a range of bytes from a binary value into this buffer, starting at index.

Throws:

An exception if index is out of range 0 .. this.length - 1 or index + value.length is out of range 0 .. this.length.

Parameters

index - Int: Zero-based starting index.

value - ByteArray: Binary value.
