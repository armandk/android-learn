toBinary

@NonNull @NotNull open static fun toBinary(@NonNull @NotNull value: @NotNull String): ByteArray

Return a binary value with simple (numeric value) mapping of source Unicode character values to target bytes. Use only when character values are limited to ISO-8859-1 (the first 256 code points of Unicode).
