JavaScript String.prototype.substr.

Parameters

value - String: Source string.

start - Int: The zero-based index at which to begin extraction from the source string. If negative, it is treated as `value.length + start`.

length - Int: Optional. The number of characters to extract. If omitted, `substr` extracts to the end of the value.

Return
String: A section of `value` beginning at `start`, with `length` characters.
