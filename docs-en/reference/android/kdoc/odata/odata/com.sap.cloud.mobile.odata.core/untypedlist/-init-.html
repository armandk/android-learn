UntypedList(capacity: Int, @Nullable @Nullable repeatValue: @Nullable Any?, repeatCount: Int)

Construct a new list with UntypedList.length of zero and specified initial capacity. A list can expand in length beyond its initial capacity, but best performance will be obtained if the initial capacity is close to (and not less than) the list's maximum length.

Parameters

capacity - Int: Initial capacity.

repeatValue - Any?: (nullable) Initial value to be repeated. Defaults to null.

repeatCount - Int: Number of repeats of initial value. Defaults to zero.
