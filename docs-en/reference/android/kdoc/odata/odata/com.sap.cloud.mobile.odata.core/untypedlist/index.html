UntypedList

open class UntypedList

An untyped list, intended for internal use by strongly-typed lists which inherit from the ListBase class.

Construct a new list with UntypedList.length of zero and specified initial capacity. A list can expand in length beyond its initial capacity, but best performance will be obtained if the initial capacity is close to (and not less than) the list's maximum length.
