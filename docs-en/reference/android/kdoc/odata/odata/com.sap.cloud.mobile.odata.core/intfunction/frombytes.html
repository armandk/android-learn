fromBytes

open static fun fromBytes(a: Int, b: Int, c: Int, d: Int): Int

Return int value converted from bytes a (high) to d (low). Each of the parameter bytes can be either signed or unsigned, and will be truncated to 8 bits if necessary

Parameters

b - Int: Second byte.

c - Int: Third byte.

d - Int: Fourth (low-order) byte.

Return
Int: Int value converted from bytes a (high) to d (low). Each of the parameter bytes can be either signed or unsigned, and will be truncated to 8 bits if necessary
