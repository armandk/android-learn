open class BinarySemaphore

A binary semaphore. A notable difference from com.sap.cloud.mobile.odata.core.ReentrantMutex is that a binary semaphore can be acquired by one thread and released by another.

Acquire the semaphore, blocking if needed until a permit is available.

release

 open fun release(): Unit

Release the semaphore which was acquired earlier using BinarySemaphore.acquire or BinarySemaphore.tryAcquire. Note: the thread that releases a semaphore need not be the same thread that acquired it.

tryAcquire

 open fun tryAcquire(timeout: Int): Boolean

Return true if the semaphore was acquired without blocking for longer than timeout milliseconds, otherwise false (in which case the semaphore was not acquired).
