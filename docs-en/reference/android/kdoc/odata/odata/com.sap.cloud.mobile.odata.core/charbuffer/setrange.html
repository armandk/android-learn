fun setRange(index: Int, @NonNull @NotNull value: @NotNull String): Unit

Copy a range of characters from a string value into this buffer, starting at index.

Throws:

An exception if index is out of range 0 .. this.length - 1 or index + value.length is out of range 0 .. this.length.

Parameters
