SparseIndexMap

open class SparseIndexMap : IntArray

Array from Property.id to sparse index. Used to implement sparse arrays for memory-optimized StrutureBase property storage. For example, a sparse array may be used when a query returning many entities only populates a few properties of each entity, even though the entity type has many properties.
