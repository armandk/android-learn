getReporter

@NonNull open fun getReporter(@NonNull targetId: String): UsageReporter

Returns a Usage Reporter with the given target ID. If the usage reporter never explicitly registered via #registerReporter(UsageReporter), a new Usage Reporter will be created and enabled, same effect as if the user has called #registerReporter(UsageReporter).

Parameters

targetId - String: target ID of the Usage Reporter

Return
UsageReporter: A usage reporter for the given target ID.
