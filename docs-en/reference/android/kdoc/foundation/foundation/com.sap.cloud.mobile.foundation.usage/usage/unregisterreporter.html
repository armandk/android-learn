unregisterReporter

open fun unregisterReporter(@NonNull targetId: String): Unit

Unregisters the previously registered custom UsageReporter. After un-registering a reporter it can no longer report to the usage store and must be re-registered to use again.

Parameters

targetId - String: a non-null target identifier associated with the UsageReporter that was previously registered via #registerReporter(UsageReporter)
