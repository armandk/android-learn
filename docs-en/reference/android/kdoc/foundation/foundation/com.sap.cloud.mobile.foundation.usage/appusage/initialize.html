initialize

static fun initialize(@NonNull context: Context, @NonNull storeName: String, @NonNull settingsParameters: SettingsParameters, @Nullable encryptionKey: ByteArray?): Unit

Initializes the mobile services usage.

Parameters

context - Context: Android application context

storeName - String: the name of the usage record persistence store

settingsParameters - SettingsParameters: Application ID, device ID, and application version will be retrieved from this object. If the application version is null, "" is used. If the device ID is null or empty, a GUID will be used.

encryptionKey - ByteArray?: the key to encrypt the underlying persistence store. It is recommended to use EncryptionUtil to obtain the encryption key. If this is the first time store is being opened and null is provided, an encryption key will be generated transparently and will be used for subsequent opens.

Exceptions

IllegalArgumentException - if the store name is empty.

OpenFailureException - if the usage record store is not opened successfully possibly due to incorrect encryption key or failed to generate the encryption (when encryptionKey is null).
