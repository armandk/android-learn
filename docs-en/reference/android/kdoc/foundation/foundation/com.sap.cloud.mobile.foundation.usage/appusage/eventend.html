eventEnd

static fun eventEnd(@NonNull targetId: String, @Nullable type: String?, @Nullable key: String?, @NonNull info: AppUsageInfo): Unit

Reports an Event#EVENT_END event for the system default targetID.

Note that before calling this method, AppUsage#eventStart(String, String, AppUsageInfo) (or one of its variant) must be called, otherwise, this EVENT_END will be discarded.

The time between the reported EVENT_START and EVENT_END will be calculated as the duration.

Parameters

targetId - String: target ID of a usage reporter

type - String?: type of the event

key - String?: event key. Will be set to "other" if null or empty.

info - AppUsageInfo: a map of additional attributes that should be reported for the event
