changeEncryptionKey

static fun changeEncryptionKey(@Nullable newEncryptionKey: ByteArray?): Unit

Changes the encryption key of the store while the store is open.

Parameters

newEncryptionKey - ByteArray?: new encryption key. It is recommended to use EncryptionUtil to obtain the new encryption key. If null is provided, an encryption key will be generated transparently and will be used for subsequent opens.

Exceptions

FileClosedException - if the persistence store has been closed.

EncryptionError - if failed to generate the encryption key (when newEncryptionKey is null).
