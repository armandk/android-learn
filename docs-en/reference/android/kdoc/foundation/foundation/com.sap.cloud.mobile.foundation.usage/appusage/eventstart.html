eventStart

static fun eventStart(@NonNull targetId: String, @Nullable type: String?, @Nullable key: String?, @NonNull info: AppUsageInfo): Unit

Reports an Event#EVENT_START event for the system default targetID.

Note that the user needs to call AppUsage#eventEnd(String, String, String, AppUsageInfo) (or its variant) with the same key, and the time between the reported EVENT_START and EVENT_END will be calculated as the duration.

If eventEnd method is not called, this EVENT_START will be ignored.

Parameters

targetId - String: target ID of a usage reporter

type - String?: type of the event

key - String?: event key. Will be set to "other" if null or empty.

info - AppUsageInfo: a map of additional attributes that should be reported for the event
