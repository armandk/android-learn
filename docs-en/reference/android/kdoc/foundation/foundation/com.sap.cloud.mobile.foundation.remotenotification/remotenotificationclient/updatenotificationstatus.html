open fun updateNotificationStatus(@NonNull notificationId: String, @NonNull status: String, @NonNull callbackListener: RemoteNotificationClient.CallbackListener): Unit

Updates the notification status at the SAP mobile cloud. Example for status are "received", "consumed" and "confirmed"

Parameters

notificationId - String: Unique notification identifier contained the in the notification message payload

status - String: The status to which the notification should be set to. Currently possible status are "received" and "consumed"

callbackListener - RemoteNotificationClient.CallbackListener: handle to CallbackListener to handle successful response or error 

Example Usage:
                              
     
                              remoteNotificationClient.updateNotificationStatus(
                                                                      notificationId,"received",
                                   new RemoteNotificationClient.CallbackListener() {
                                       @Override
                                       public void onSuccess() {
                                       //Here goes the code for processing successful response...
                                   }
     
                                     @Override
                                     public void onError(Throwable result) {
                                        //Handle error here...
                                      }
     
                              });
