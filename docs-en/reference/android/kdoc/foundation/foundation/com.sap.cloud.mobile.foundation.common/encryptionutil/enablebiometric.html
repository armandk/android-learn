foundation / com.sap.cloud.mobile.foundation.common / EncryptionUtil / enableBiometric

enableBiometric

open static fun enableBiometric(@NonNull alias: String, @NonNull passcode: CharArray, @NonNull cipher: Cipher): Unit

When in NO_PASSCODE or PASSCODE_ONLY state, re-encrypted the encryption key with the passcode and biometric authentication.

Parameters

alias - String: the alias of the encryption key

passcode - CharArray: the existing passcode when in PASSCODE_ONLY state or a new passcode when in NO_PASSCODE state

cipher - Cipher: the cipher returned from #getCipher(String)

Exceptions

IllegalStateException - If the current encryption state is PASSCODE_BIOMETRIC already.

EncryptionError - If the passcode is incorrect when the current state is PASSCODE_ONLY or if encryption/decryption error occurs.
