EncryptionOutputStream(@NonNull out: OutputStream, encrypt: Boolean, @NonNull encryptionKey: ByteArray)

Parameters

out - OutputStream: The OutputStream that will be encrypted/decrypted.

encrypt - Boolean: True means the OutputStream will be encrypted. False means the OutputStream will be decrypted.

encryptionKey - ByteArray: The key which is used for encryption/decryption. It is recommended to use EncryptionUtil to generate the encryption key, but this is not required. The encryption key must be 16, 24, or 32 bytes. When the application no longer needs the encryption key it should clear the array with a call like: Arrays.fill(encryptionKey, (byte) 0x0);
