encryptStream

open static fun encryptStream(@NonNull inputStream: InputStream, @NonNull outputStream: OutputStream, @NonNull encryptionKey: ByteArray): Unit

Encrypts the contents of a stream and passes the encrypted contents to another stream. This method can be used anywhere streams are used. For example, a file can be encrypted by passing this method a FileInputStream and a FileOutputStream. This method does not close any streams, that is the responsibility of the caller.

Parameters

inputStream - InputStream: The stream where unencrypted bytes will be read from. This stream will be closed after calling this method.

outputStream - OutputStream: The stream where encrypted bytes will be written to. This stream will be closed after calling this method.

encryptionKey - ByteArray: The key used to encrypt the bytes, and which can be used to decrypt the encrypted bytes later. It is recommended to use EncryptionUtil to generate the encryption key, but this is not required. The encryption key must be 16, 24, or 32 bytes. When the application no longer needs the encryption key it should clear the array with a call like: Arrays.fill(encryptionKey, (byte) 0x0);

Exceptions

IOException - Indicates something went wrong with IO or encrypting.
