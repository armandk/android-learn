decrypt

@NonNull open static fun decrypt(@NonNull encryptedData: ByteArray, @NonNull encryptionKey: ByteArray): ByteArray

Decrypts the contents of the given byte array with the given encryption key.

Parameters

encryptedData - ByteArray: The encrypted bytes.

encryptionKey - ByteArray: The key used to decrypt the bytes. This must be the same key that was used to encrypt the bytes, or else an exception will be thrown. When the application no longer needs the encryption key it should clear the array with a call like: Arrays.fill(encryptionKey, (byte) 0x0);

Exceptions

IOException - Indicates something when wrong with decryption.

Return
ByteArray: The decrypted bytes.
