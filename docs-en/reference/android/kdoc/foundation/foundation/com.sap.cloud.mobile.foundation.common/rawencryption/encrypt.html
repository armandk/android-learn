encrypt

@NonNull open static fun encrypt(@NonNull data: ByteArray, @NonNull encryptionKey: ByteArray): ByteArray

Encrypts the contents of the given byte array with the given encryption key.

Parameters

data - ByteArray: The unencrypted bytes.

encryptionKey - ByteArray: The key that will be used to encrypt the data, and can be used later to decrypt the encrypted bytes. It is recommended to use EncryptionUtil to generate the encryption key, but this is not required. The encryption key must be 16, 24, or 32 bytes. When the application no longer needs the encryption key it should clear the array with a call like: Arrays.fill(encryptionKey, (byte) 0x0);

Exceptions

IOException - Indicates something went wrong with encryption.

Return
ByteArray: The encrypted bytes.
