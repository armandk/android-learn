decryptFile

open static fun decryptFile(@NonNull encryptedFilePath: String, @NonNull decryptedFilePath: String, @NonNull encryptionKey: ByteArray): Unit

Decrypts a file and saves the decrypted contents to another file.

Parameters

encryptedFilePath - String: The complete filepath to the encrypted file.

decryptedFilePath - String: The complete filepath to where the decrypted file should be written. If there is already a file at this filepath it will be overwritten.

encryptionKey - ByteArray: The key used to decrypt the file, and which can be used to decrypt the file later. It is recommended to use EncryptionUtil to generate the encryption key, but this is not required. The encryption key must be 16, 24, or 32 bytes. When the application no longer needs the encryption key it should clear the array with a call like: Arrays.fill(encryptionKey, (byte) 0x0);

Exceptions

IOException - Indicates something went wrong with IO or decryption.
