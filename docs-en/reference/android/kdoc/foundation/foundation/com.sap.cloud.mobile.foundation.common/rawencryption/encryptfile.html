encryptFile

open static fun encryptFile(@NonNull unencryptedFilePath: String, @NonNull encryptedFilePath: String, @NonNull encryptionKey: ByteArray): Unit

Encrypts a file and saves the encrypted contents to another file.

Parameters

unencryptedFilePath - String: The complete filepath to the original, unencrypted file.

encryptedFilePath - String: The complete filepath to where the encrypted file should be written. If there is already a file at this filepath it will be overwritten.

encryptionKey - ByteArray: The key used to encrypt the file, and which can be used to decrypt the file later. It is recommended to use EncryptionUtil to generate the encryption key, but this is not required. The encryption key must be 16, 24, or 32 bytes. When the application no longer needs the encryption key it should clear the array with a call like: Arrays.fill(encryptionKey, (byte) 0x0);

Exceptions

IOException - Indicates something went wrong with IO or encryption.
