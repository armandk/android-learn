SDKApplicationState()

Represents the application state.

When initializing the mobile servcies with com.sap.cloud.mobile.foundation.mobileservices.MobileServices, the services will response on the current application state, for example, when the application is foreground, the usage service can start the session to record, then end session when application goes to background. The instance of this class will be responsible for this purpose.
