put

@Nullable fun put(@NonNull key: K, @NonNull value: V, cost: Double): V?

Cost Factor feature is not supported by Composite Cache.

Parameters

key - K: a non-null key of a cache entry

value - V: a non-null value to add

cost - Double: the cost associated with this entry. If the accumulated cost exceeds maximum cache cost, cache entry eviction may be triggered to free up memory space.

Exceptions

UnsupportedOperationException - Cost Factor is not supported for Composite Cache.

Return
V?: The previous value associated with the existed key, null if the entry did exist.

@Nullable fun put(@NonNull key: K, @NonNull value: V): V?

Adds an entry to the composite cache. If the value is the same as original value, nothing will happen. Otherwise, the entry will be populated from top to bottom through cache levels.

Parameters

key - K: key of the cache entry

value - V: value of the cache entry

Return
V?: The previous value associated with the key or null if the entry did not exist before.
