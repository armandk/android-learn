add

@NonNull fun <T : Cache<K, V>!> add(@NonNull cache: T): CompositeCache<K, V>

Adds a concrete cache instance as one level of cache in the composite cache. Depending on the level of caches desired, this method needs be called in the order of cache levels until #build() is called.

Parameters

cache - T: an instance of non-null concrete Cache class

Exceptions

IllegalStateException - if CompositeCache#build() has been called already

Return
CompositeCache<K, V>: This composite cache instance, which can be used to call this method again or complete the composite cache initialization via #build().
