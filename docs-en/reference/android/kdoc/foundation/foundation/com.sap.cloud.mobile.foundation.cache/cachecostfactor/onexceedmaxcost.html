onExceedMaxCost

@NonNull abstract fun onExceedMaxCost(@NonNull existingKeys: MutableList<K>): MutableList<K>

This method is invoked during Cache#put(Object, Object, double) to instruct the cache implementation to remove one or more cache items from a list of existing keys so that a new cache item can be added when the aggregated costs exceeds the maximum cost.

  1. The cache will remove the entry/entries based on the returned list.
  2. The current Cache#put(Object, Object, double) will proceed and the entry will be added successfully.

Parameters

existingKeys - MutableList<K>: a list of keys associated with existing cache entries sorted in cost ascending order

Return
MutableList<K>: A non-null list of keys selected from the list of existing keys.
