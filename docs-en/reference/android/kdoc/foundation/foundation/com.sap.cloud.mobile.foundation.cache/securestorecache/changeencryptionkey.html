changeEncryptionKey

fun changeEncryptionKey(@Nullable newEncryptionKey: ByteArray?): Unit

Changes the encryption key of the persistence store while the cache is in open state.

Parameters

newEncryptionKey - ByteArray?: new encryption key. It is recommended to use EncryptionUtil to obtain the new encryption key. If null is provided, an encryption key will be generated transparently and will be used for subsequent opens.

Exceptions

EncryptionError - if failed to generate the encryption key transparently when newEncryptionKey is null or zero length.

IllegalStateException - if the store is not in open state.
