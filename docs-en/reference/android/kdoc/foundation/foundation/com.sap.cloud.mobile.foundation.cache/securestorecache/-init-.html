SecureStoreCache(@NonNull context: Context, maxEntries: Int, @NonNull storeName: String)

Constructs a Secure Store Cache with LRU replacement policy.

Parameters

context - Context: Android application context

maxEntries - Int: maximum number of entries the cache can hold. Uses internal default size if the value <= 0.

storeName - String: a non-empty name of the persistence store that contains only -, _, a-z, A-Z, 0-9 and .. The store will be created if it does not exist.

Exceptions

IllegalArgumentException - when persistence store name is empty or invalid.
