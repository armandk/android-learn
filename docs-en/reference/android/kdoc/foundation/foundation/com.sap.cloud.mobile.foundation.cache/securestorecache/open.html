open

fun open(@Nullable encryptionKey: ByteArray?): Unit

Opens the underlying persistence store.

Parameters

encryptionKey - ByteArray?: a key to encrypt/decrypt the persistence store. It is recommended to use EncryptionUtil to obtain the encryption key. If this is the first time store is being opened and null is provided, an encryption key will be generated transparently and will be used for subsequent opens.

Exceptions

OpenFailureException - failed to open the store due to incorrect encryption key or failed to generate the encryption key.
