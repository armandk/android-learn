put

@Nullable fun put(@NonNull key: String, @NonNull value: V, cost: Double): V?

Not Supported.

Parameters

key - String: a non-null key of a cache entry

value - V: a non-null value to add

cost - Double: the cost associated with this entry. If the accumulated cost exceeds maximum cache cost, cache entry eviction may be triggered to free up memory space.

Exceptions

UnsupportedOperationException - always; cost factor is not supported.

Return
V?: The previous value associated with the existed key, null if the entry did exist.
