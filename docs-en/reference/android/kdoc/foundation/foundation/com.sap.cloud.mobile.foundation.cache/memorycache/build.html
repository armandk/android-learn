build

@NonNull fun build(): MemoryCache<K, V>

Completes the Memory Cache instance configuration. The cache CRUD operations are only then enabled after returning from this call.

1. The cache size have the maximum entries specified via #MemoryCache(Context, int).

2. The cache will evict all cache entries if #clearOnMemoryError() has been called.

3. When Cost Factor mechanism is enabled: 


  a. No customized CacheCostFactor#onExceedMaxCost(List) is provided
     (via #addCostFactor(CacheCostFactor)), a system default method will be invoked to
     remove the most costly entry amongst the existing entries during a call to
     #put(Object, Object, double), then proceeds with the original put operation.
  b. Otherwise, the customized CacheCostFactor#onExceedMaxCost(List) will be invoked to
     remove the selected entry/entries before proceeding with the put operation.
