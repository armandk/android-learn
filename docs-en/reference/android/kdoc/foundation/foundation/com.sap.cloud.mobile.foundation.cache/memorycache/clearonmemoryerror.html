clearOnMemoryError

@NonNull fun clearOnMemoryError(): MemoryCache<K, V>

Sets a flag so that when low memory condition occurs, all cache entries will be removed(clearing the cache).

The clear on low memory action is taken when:

1. The application component implemented ComponentCallbacks#onLowMemory() or ComponentCallbacks2#onLowMemory(), and called #onLowMemory().

2. OutOfMemoryError is detected inside Memory Cache, and #onLowMemory() will be invoked internally.

Exceptions

IllegalStateException - if #build() has been called

Return
MemoryCache<K, V>: The same cache instance this method was called upon.
