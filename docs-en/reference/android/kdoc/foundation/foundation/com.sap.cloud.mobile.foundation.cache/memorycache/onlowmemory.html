fun onLowMemory(): Unit
fun onLowMemory(): Unit

When the system is running low on memory, evicts all entries in the cache if #clearOnMemoryError() was called during Memory Cache initialization.

This method should be called from Android component's ComponentCallbacks#onLowMemory() or android.content.ComponentCallbacks2#onTrimMemory(int), and the system will perform a garbage collection after returning from that method.

Exceptions

IllegalStateException - if the cache is not yet configured.
