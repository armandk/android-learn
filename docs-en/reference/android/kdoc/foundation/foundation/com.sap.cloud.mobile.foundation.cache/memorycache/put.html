@Nullable fun put(@NonNull key: K, @NonNull value: V, cost: Double): V?

Adds the value and the associated cost to the cache for the key. Only valid if #maxCost(double) was called.

Parameters

key - K: non-null key of the cache entry

value - V: non-null value of the cache entry

cost - Double: a cost > 0 associated with this entry

Exceptions

IllegalStateException - if Cost Factor is not enabled, i.e., #maxCost(double) was not called.

IllegalArgumentException - if cost < 0.

Return
V?: The previous value associated with the existed key, null otherwise.
