addCostFactor

@NonNull fun <T : CacheCostFactor<K>!> addCostFactor(@NonNull costFactor: T): MemoryCache<K, V>

Assuming Cost Factor mechanism has been enabled via #maxCost(double), this method provides customized cache replacement method #maxCostExceeded(double, double) when the aggregated cost exceeds the maximum cost during #put(Object, Object, double).

Parameters

costFactor - T: An instance of CacheCostFactor

Exceptions

IllegalStateException - if the #build() has been called, or #maxCost(double) has not been called to enable the Cost Factor mechanism

Return
