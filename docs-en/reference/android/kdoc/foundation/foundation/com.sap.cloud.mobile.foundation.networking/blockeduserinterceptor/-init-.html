BlockedUserInterceptor(@NonNull blockedUserCallback: BlockedUserInterceptor.BlockedUserCallback)

Creates a blocked user interceptor object which will fire the specified callback in the case that an http request is made and the user is blocked at the server.

Parameters

blockedUserCallback - BlockedUserInterceptor.BlockedUserCallback: A BlockedUserCallback implementation that will be invoked when an http request is made and the user is blocked by the server. Note: that will be invoked synchronously so it should return as quickly as possible, since it will be holding up the http request call from completing.
