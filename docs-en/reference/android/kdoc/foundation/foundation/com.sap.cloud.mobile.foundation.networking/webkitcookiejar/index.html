WebkitCookieJar

open class WebkitCookieJar : CookieJar
open class WebkitCookieJar : CookieJar

This class implements okhttp3.CookieJar by storing/getting all cookies with android.webkit.CookieManager. If an application needs to do any custom cookie handling (eg: clearing session cookies), that must be done on the underlying android.webkit.CookieManager.
