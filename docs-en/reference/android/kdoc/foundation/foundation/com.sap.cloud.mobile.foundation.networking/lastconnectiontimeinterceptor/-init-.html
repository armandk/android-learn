LastConnectionTimeInterceptor(@NonNull lastConnectionTimeCallback: LastConnectionTimeInterceptor.LastConnectionTimeCallback)

Creates a last connection time interceptor object which will fire the specified callback whenever a successful connection with the server occurs.

Parameters

lastConnectionTimeCallback - LastConnectionTimeInterceptor.LastConnectionTimeCallback: A LastConnectionTimeCallback implementation that will be invoked whenever a successful connection with the server occurs so that the persisted last connection time can be updated. Note: that will be invoked synchronously so it should return as quickly as * possible, since it will be holding up the http request call from completing.
