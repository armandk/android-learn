update

abstract fun update(bytesRead: Long, percentComplete: Int): Unit

This function is called with progress information for a response. This function only gets invoked once for each percent.

Parameters

bytesRead - Long: The total number of bytes read.

percentComplete - Int: The percent complete is always rounded down, so when this value is 100 all bytes have been read (though the stream might not be closed).
