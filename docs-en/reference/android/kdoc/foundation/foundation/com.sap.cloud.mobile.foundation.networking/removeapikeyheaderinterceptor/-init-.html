RemoveApiKeyHeaderInterceptor()
Deprecated: As of release 3.3

In case of Basic authentication, the authenticator kicks in only after getting 401 response. But in the presence of api key, the request will pass through and it will never return 401. So the api key header should be removed before sending Basic authentication request.
