CsrfTokenInterceptor(@NonNull urlProvider: CsrfTokenUrlProvider)

Constructs a new CsrfTokenInterceptor. Use if you have multiple backends and the backends accept different CSRF tokens. By default the tokens will be stored by host and port in memory.

Parameters

urlProvider - CsrfTokenUrlProvider: The CsrfTokenUrlProvider implementation which provides the CSRF URL for every request URL which are protected with CSRF.
