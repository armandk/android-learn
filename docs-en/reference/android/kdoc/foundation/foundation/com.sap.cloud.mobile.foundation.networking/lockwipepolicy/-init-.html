LockWipePolicy(enabled: Boolean, lockDays: Int, wipeDays: Int)

Creates a lock & wipe policy object with no callbacks specified.

Parameters

enabled - Boolean: Specifies whether the entire policy is enabled or not.

lockDays - Int: The number of 24 hour periods that need to elapse without a successful connection to the server in order to lock the application. May be zero to indicate that there is no check for locking.

wipeDays - Int: The number of 24 hour periods that need to elapse without a successful * connection to the server in order to wipe the application. May be zero to indicate that there is no check for wiping.
