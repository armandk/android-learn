AuthenticationUiCallbackManager

open class AuthenticationUiCallbackManager
open class AuthenticationUiCallbackManager

This class provides a place to set and call the AuthenticationUiCallback that decides whether to allow UI to be shown to authenticate the user. #setAuthenticationUiCallback(AuthenticationUiCallback) sets the callback that will be used. #allowShowingUiToAuthenticate() will return the value returned by AuthenticationUiCallback#allowShowingUiToAuthenticate(). If the callback has not been set, or has been set to null, a default implemenation of AuthenticationUiCallback is used that always returns true.
