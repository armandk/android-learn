authenticate

@WorkerThread @Nullable open fun authenticate(): OAuth2Token?

Starts the OAuth 2.0 authentication process. When using the Authorization Code Grant type this method will present a web view.

Exceptions

IOException - if the request could not be executed due to cancellation, a connectivity problem or timeout.

Return
OAuth2Token?: The obtained OAuth2Token if the process was successful, otherwise null.
