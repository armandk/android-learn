allowShowingUiToAuthenticate

abstract fun allowShowingUiToAuthenticate(): Boolean

Return
Boolean: true if the application is in a state where it is OK to show authentication UI (eg: a login page). false if the application would rather the request getting the authentication challenge fails instead of showing UI to authenticate.
