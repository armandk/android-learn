abstract fun storeCredential(@NonNull rootUrl: String, @NonNull realm: String, @NonNull credentials: Array<String!>): Unit

Stores credentials for future use.

Parameters

rootUrl - String: The root URL for which the credentials should be used for. The root URL is composed of the scheme, host, and port.

realm - String: The realm for which the credentials should be used for.

credentials - Array<String!>: This array should have a length of 2. The first index should have the username, the second index should have the password.
