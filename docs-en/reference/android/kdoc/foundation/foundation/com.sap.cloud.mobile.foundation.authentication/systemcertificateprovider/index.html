SystemCertificateProvider

open class SystemCertificateProvider : CertificateProvider
open class SystemCertificateProvider : CertificateProvider

An implementation of the CertificateProvider that provides certificates available in the android.security.KeyChain. The first time a certificate is requested for a given host and port, this class will launch the Android UI necessary for the app to get permission to use the certificate the user selects. The user selection is saved in the AliasStore (if provided) and the certificate picker UI will not be shown again unless the AliasStore has that certificate cleared. Since different servers might require different credentials, the certificate selection is saved per host/port combination.
