A new instance of BasicAuthInMemoryStore will be used by default to store credentials for reuse.

BasicAuthDialogAuthenticator(@Nullable basicAuthCredentialStore: BasicAuthCredentialStore?)

Allows a non-default BasicAuthCredentialStore to be specified.

Parameters

basicAuthCredentialStore - BasicAuthCredentialStore?: This store is used so that credentials can be reused on subsequent challenges without prompting the user. A new instance of BasicAuthInMemoryStore will be used if this parameter is null.

BasicAuthDialogAuthenticator(@Nullable basicAuthCredentialStore: BasicAuthCredentialStore?, allowInsecureBasicAuth: Boolean)

Allows options to be specified.

Parameters

basicAuthCredentialStore - BasicAuthCredentialStore?: This store is used so that credentials can be reused on subsequent challenges without prompting the user. A new instance of BasicAuthInMemoryStore will be used if this parameter is null.

allowInsecureBasicAuth - Boolean: Allows the constructed BasicAuthDialogAuthenticator to perform basic authentication over an insecure connection (HTTP). This will broadcast the credentials used to any eavesdroppers and SHOULD NOT BE SET TO TRUE except with test credentials in a test environment for debugging purposes.
