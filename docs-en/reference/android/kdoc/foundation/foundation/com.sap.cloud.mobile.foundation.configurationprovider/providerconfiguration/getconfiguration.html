getConfiguration

@Nullable open fun getConfiguration(): JSONObject?

Retrieves the configuration data as a JSONObject configuration object.

Return
JSONObject?: A JSONObject configuration object containing the configuration data or null is there was an error. The JSONObject configuration will be empty if the provider could not retrieve the configuration data, but did not encounter an input-related error.
