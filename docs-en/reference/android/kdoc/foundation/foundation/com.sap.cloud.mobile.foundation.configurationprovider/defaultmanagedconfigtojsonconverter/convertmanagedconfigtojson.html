convertManagedConfigToJson

@Nullable open fun convertManagedConfigToJson(@Nullable appRestrictions: Bundle?, @Nullable manifestRestrictions: MutableList<RestrictionEntry!>?): JSONObject?

Converts managed configuration data to json.

Parameters

appRestrictions - Bundle?: a Bundle representing managed configuration data that is to be converted to JSON.

manifestRestrictions - MutableList<RestrictionEntry!>?: a list of the application restriction (Managed Configuration) entries that are in the application manifest.

Return
JSONObject?: a JSONObject representing the configuration information contained in the appRestrictions. If an error is encountered, null is returned and the last error is set to describe the error.
