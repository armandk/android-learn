parseProviderIdentifier

@Nullable open static fun parseProviderIdentifier(@NonNull identifierString: String, @NonNull knownProviders: Array<ProviderIdentifier!>): ProviderIdentifier?

Return the provider corresponding to the supplied provider identifier string, based on the supplied list of known provider identifiers. If custom providers are added, the list of known providers must come from the ProviderIdentifier sub-class.

Parameters

identifierString - String: A provider identifier string.

knownProviders - Array<ProviderIdentifier!>: A list of all of the known provider identifiers, including custom providers.

Return
ProviderIdentifier?: The provider corresponding to the supplied provider identifier string or null if it does not match a known provider idewntifier.
