store

open fun store(@NonNull keyPath: String, @NonNull jsonObject: JSONObject, @NonNull callbackListener: Settings.CallbackListener): Unit

Store the settings value at the server. If value is already present, it will be updated. If it is not existing it will be created. This function is equivalent to #store(SettingTarget, String, JSONObject, CallbackListener), but the SettingTarget is assumed to be SettingTarget#DEVICE

Parameters

keyPath - String: is the slash(/) separated key values specifying the level where the values need to be stored

jsonObject - JSONObject: the JSONObject value to be set.

callbackListener - Settings.CallbackListener: instance of CallbackListener to process result or handle error 

Example Usage:
                                                                                                      
     
                                                                                                      Settings settings = new Settings(getOkHttpClientForSSL(), uri, null, null);
                                                                                                      JSONObject jsonVal = null;
                                                                                                      try {
                                                                                                        jsonVal =new JSONObject("{'a':'b'}");
                                                                                                      } catch (JSONException e) {
                                                                                                        //Handle your exception here
                                                                                                      }
                                                                                                      settings.store(SettingTarget.APPLICATION
                                                                                                         ,"path/to/location",jsonVal,
                                                                                                         new Settings.CallbackListener()() {
                                                                                                           @Override
                                                                                                            public void onSuccess(JSONObject o) {
                                                                                                            //Here goes the code for processing successful response...
                                                                                                            }
     
                                                                                                             @Override
                                                                                                            public void onError(Throwable result) {
                                                                                                                //Handle error here...
                                                                                                            }
     
                                                                                                      });
