open fun load(@Nullable keyPath: String?, @NonNull callback: Settings.CallbackListener): Unit

Reads the settings value from the server. The default target is SettingTarget#DEVICE. This is an asynchronous function.

Parameters

keyPath - String?: string value denoting the path to the key to load

callback - Settings.CallbackListener: a CallbackListener instance to handle response 

Example Usage:
                                                                      
     
                                                                      final Settings settings = new Settings(getOkHttpClientForSSL(), uri, null, null);
                                                                       settings.load(
                                                                        new Settings.CallbackListener() {
                                                                         @Override
                                                                            public void onSuccess(JSONObject o) {
                                                                            //Here goes the code for processing successful response...
                                                                            }
     
                                                                             @Override
                                                                            public void onError(Throwable result) {
                                                                                //Handle error here...
                                                                            }
     
                                                                      });
