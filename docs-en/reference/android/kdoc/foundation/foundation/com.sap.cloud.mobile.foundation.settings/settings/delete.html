delete

open fun delete(@Nullable keyPath: String?, @NonNull callbackListener: Settings.CallbackListener): Unit

Issues a request to the server to delete the key value specified by settingKeyPath. This function is equivalent to #delete(SettingTarget, String, CallbackListener) is assumed to be SettingTarget#DEVICE.

Parameters

keyPath - String?: is the slash(/) separated key values specifying the level where the value need to be set

callbackListener - Settings.CallbackListener: instance of CallbackListener to process result or handle error 

Example Usage:
                                                                                                      
     
                                                                                                      final Settings settings = new Settings(getOkHttpClientForSSL(), uri, null, null);
                                                                                                      //You can use a specific path or use a constant provided in Settings.KeyPath
                                                                                                      settings.delete("path/to/location",
                                                                                                          new Settings.CallbackListener()() {
                                                                                                         @Override
                                                                                                            public void onSuccess(JSONObject o) {
                                                                                                            //Here goes the code for processing successful response...
                                                                                                            }
     
                                                                                                             @Override
                                                                                                            public void onError(Throwable result) {
                                                                                                                //Handle error here...
                                                                                                            }
     
                                                                                                      });
