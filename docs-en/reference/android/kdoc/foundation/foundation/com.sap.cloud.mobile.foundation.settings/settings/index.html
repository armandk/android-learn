SAP cloud mobile Settings class provides an API interface to get the application connection settings from the server and also to update these settings. This class is initialized with the information about an application url for the settings information. The settings object can be considered as a key value repository. Value can be got and set. The Settings#load and Settings#update(SettingTarget, String, JSONObject, CallbackUpdateListener) will get the value from the server and update the value at the server. The Settings#delete(String, CallbackListener) will is used to delete existing settings values. Deprecated since 3.2, replaced with ClientPolicyService
