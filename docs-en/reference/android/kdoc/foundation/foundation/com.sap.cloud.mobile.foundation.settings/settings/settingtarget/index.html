class SettingTarget

Enumeration to specify what is the settings target level to be used

  • APPLICATION: This is an application level property list. The properties are for all the devices registrations that is using this application.

  • USER: This is a user level setting. This is comment to all registrations f it is under the same user. For example a single user can be registered on multiple devices. These properties are shared among all those device registrations.

  • DEVICE: This is a device level properties.
