executeQuery

@NonNull fun executeQuery(@NonNull sql: String, @NonNull vararg values: String!): SecureDatabaseResultSet

Executes a SELECT statement that contains parameters ('?').

Parameters

sql - String: a SELECT statement

values - String!: one or more String values to match the '?' in the SQL statement

Exceptions

FileClosedException - if the database has been closed.

BackingStoreException - if the query fails.

Return
SecureDatabaseResultSet: a non-nullresult set from the query which may or may not contain rows.

@NonNull fun executeQuery(@NonNull sql: String): SecureDatabaseResultSet

Executes a SELECT statement that does not contains parameters ('?').
