SecureDatabaseStore(@NonNull context: Context, @NonNull databaseName: String, databaseVersion: Int, @NonNull cb: CreateDatabaseCallback)

Constructs a Secure Database Store and invokes the provided CreateDatabaseCallback#onCreate(SecureDatabaseStore) for creating database objects if those objects do not exist already, or invokes CreateDatabaseCallback#onUpgrade(SecureDatabaseStore, int, int) if the database version is newer than the existing database.

Note: The caller must call #open(byte[]) after a SecureDatabaseStore instance is created. Otherwise, all accesses to the instance will cause FileClosedException.

Parameters

context - Context: Android application context

databaseName - String: a non-empty name of the Database that can only contain -, _, a-z, A-Z, 0-9 and ..

cb - CreateDatabaseCallback: callback that is used to to specify table schema to create or update the database objects

Exceptions

IllegalArgumentException - If database name is empty or invalid, of the database version < 1
