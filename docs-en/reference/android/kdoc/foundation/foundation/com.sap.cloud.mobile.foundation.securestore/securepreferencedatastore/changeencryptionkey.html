foundation / com.sap.cloud.mobile.foundation.securestore / SecurePreferenceDataStore / changeEncryptionKey

changeEncryptionKey

fun changeEncryptionKey(@Nullable newEncryptionKey: ByteArray?): Unit

Changes the encryption key of the store while the store is opened (#close() is not yet called).

Parameters

newEncryptionKey - ByteArray?: new encryption key. It is recommended to use EncryptionUtil to obtain the new encryption key. If null is provided, an encryption key will be generated transparently and will be used for subsequent opens.

Exceptions

FileClosedException - if the store has been closed or failed to generate the encryption (when newEncryptionKey is null).
