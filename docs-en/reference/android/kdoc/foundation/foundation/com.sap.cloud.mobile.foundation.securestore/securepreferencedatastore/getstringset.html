getStringSet

@Nullable fun getStringSet(@NonNull key: String, @Nullable defValues: MutableSet<String!>?): MutableSet<String!>?

Retrieves a set of Strings from the data store.

Parameters

key - String: the name of the preference to retrieve

defValues - MutableSet<String!>?: the values to return if this preference does not exist in the storage

Exceptions

TypeConversionException - the Set provided is not Serializable.

FileClosedException - if the store has been closed.

BackingStoreException - if an error occurs when adding the value to the store.

Return
MutableSet<String!>?: The values from the data store or the default return values.
