getInt

fun getInt(@NonNull key: String, defValue: Int): Int

Retrieves an integer value from the data store.

Parameters

key - String: the name of the preference to retrieve

defValue - Int: the value to return if this preference does not exist in the data store

Exceptions

FileClosedException - if the store has been closed.

BackingStoreException - if an error occurs when adding the value to the store.

Return
Int: The value from the data store or the default return value.
