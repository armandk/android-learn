get

@Nullable fun <T : Any!> get(@NonNull key: String): T?

Retrieves the value associated with a key. Note that it is the caller's responsibility to assign the returned value to the same type(class) of variable as the value was #put(String, Object), otherwise, the caller will get ClassCastException. 


For example,
    final String stringKey1 = "myStringKey";
    kvStore.put(stringKey1, "myStringValue");
    ...
    Integer value = kvStore.get(stringKey1);  // Will get ClassCastException here.

Parameters

key - String: the key associated with the value

- the original type passed in during previous #put(String, Object)

Return
T?: The value associated with the key.
