fun open(@Nullable encryptionKey: ByteArray?): Unit

Opens the key value store with the encryption key.

Parameters

encryptionKey - ByteArray?: the key to encrypt the store. It is recommended to use EncryptionUtil to obtain the encryption key. If this is the first time store is being opened and null is provided, an encryption key will be generated transparently and will be used for subsequent opens.

Exceptions

OpenFailureException - if the encryption key is incorrect or failed to generate the encryption key.
