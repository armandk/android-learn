getSerializable

@Nullable fun <T : Serializable!> getSerializable(@NonNull key: String): T?

Retrieves the Serializable object for the give key.

Parameters

- the type of the Serializable object

Exceptions

FileClosedException - if the store has been closed.

TypeConversionException - if the object cannot be de-serialized.

Return
