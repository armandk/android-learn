downloadResourceToFile

fun downloadResourceToFile(listener: ServiceListener<String>, fileName: String, bundle: ClientResourceBundle? = null, overwrite: Boolean = true): Job

Downloads the client resource and save the content to a file identified by fileName, if bundle is not specified, the default client resource will be downloaded. The saved file is located in the internal storage of this mobile application, the client code can then access the file with: val file = File(context.filesDir, filename)
