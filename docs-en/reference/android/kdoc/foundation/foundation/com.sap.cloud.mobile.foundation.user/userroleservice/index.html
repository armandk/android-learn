UserRoleService

class UserRoleService : MobileService
Deprecated: Deprecated since 3.2

Represents the service to update user information automatically when users is authenticated or the mobile application is brought to foreground. To make the service automatically do this, userInfoChangeListener must be provided, and it will only notified when new user information is retrieved from mobile services, then both old user information and the new information will be notified to client code.
