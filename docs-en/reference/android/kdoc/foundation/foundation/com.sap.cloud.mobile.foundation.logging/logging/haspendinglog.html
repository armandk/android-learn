hasPendingLog

static fun hasPendingLog(): Boolean

Returns true if there is an old log that did get uploaded from a previous upload attempt. If an upload is currently in progress false is returned.

Exceptions

IllegalStateException - Thrown when logging has not been initialized or the logger context has been reset.

Return
Boolean: false if an upload is currently in progress or last call to uploadLog() completed successfully. Otherwise, returns true.
