uploadLog

static fun uploadLog(@NonNull client: OkHttpClient, @NonNull settingsParameters: SettingsParameters): Unit
Deprecated: As of release 3.0, replaced by #upload(OkHttpClient, SettingsParameters) Upload the log to mobile services. The log will be purged as part of the upload.

If there is an old log that did not get uploaded from a previous upload attempt, the old log is deleted and not uploaded. The upload is executed in the background using an AsyncTask using AsyncTask.THREAD_POOL_EXECUTOR.

Parameters

client - OkHttpClient: Client that has the mobile services session already established.

settingsParameters - SettingsParameters: The application's setting parameters.

Exceptions

IllegalStateException - Thrown when logging has not been initialized or the logger context has been reset, or when an upload is already in progress.
