logFileSize

@NonNull open fun logFileSize(@NonNull logFileSize: String): Logging.ConfigurationBuilder

Set the log file size.

Parameters

logFileSize - String: The size of each log file. It is formatted as an integer quantity followed by upper case character units (KB, MB). For example, "100KB". Note that max total log size is logFileCount times logFileSize.

Exceptions

IllegalArgumentException - when invalid parameters are passed

Return
