class UserSecureStoreDelegate
Content copied to clipboard

Represents the user secure store delegate which helps client code retrieving/saving data into the passcode protected secure store.

This delegate can only be used after onboarding or restore flow is done, otherwise, runtime exception will be encountered. And there are some system protected keys which don't allow client code to access, even for read due to security reasons. If client code happens to use the same key as system keys, errors will be reported.
