FlowStateListener

abstract class FlowStateListener
Content copied to clipboard

Represents the flow state listener, with which client code can insert logic at certain points during the flow process, for example, after passcode is created, client code can use that to encrypt a local secure store.

All callbacks defined here will be executed in the IO thread except onFlowFinished which be executed in Main thread.

For callback functions of onUserSwitched, onOfflineEncryptionKeyReady and onFlowFinished, they will be notified to client in the above sequence.

When onUserSwitched is notified, FlowContext.getCurrentUserId and FlowContext.getPreviousUserId will also be available, so they can be used in onOfflineEncryptionKeyReady and onFlowFinished to know whether a user switch happened or not.
