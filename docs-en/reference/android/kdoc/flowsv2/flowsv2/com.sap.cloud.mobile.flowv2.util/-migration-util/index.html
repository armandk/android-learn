class MigrationUtil

Represents the migration utility.

To use this utility, the mobile app needs to unlock the app and make 'okhttp' ready in 'ClientProvider', so the utility can get the user detail and the passcode policy from the server side.

The migration process will NOT enable biometric, if client code is running in the single user mode, the app can guide user to enable biometric by running the change passcode flow.
